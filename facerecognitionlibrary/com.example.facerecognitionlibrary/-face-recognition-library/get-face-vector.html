getFaceVector

fun getFaceVector(context: Context, byteArr: ByteArray): DoubleArray?

Extract the face feature vector from the passed ByteArray data. return face feature vector as a DoubleArray, or null if the extraction fails. The ByteArray data must contain a face image.

Return

The extracted face feature vector as a DoubleArray, or null if the extraction fails.

Parameters

context

The context of the Android application.

byteArr

The ByteArray data containing the face image.
