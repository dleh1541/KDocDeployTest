FaceRecognitionLibrary

abstract class FaceRecognitionLibrary(context: Context, activity: AppCompatActivity, threshold: Int = 10)

Abstract class representing a face recognition library. To use this library, you need to implement the abstract function reachSeletedCount(), which defines the action to take when face extraction is successful.

Parameters

context

The context of the Android application.

activity

The reference to the current activity or AppCompatActivity.

threshold

The threshold value for face detection count. Default value is 10.
